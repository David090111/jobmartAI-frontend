# JobMart AI

JobMart AI is a full-stack platform for job posting, product selling, and career guidance powered by AI. It is built to help users connect with job opportunities, showcase their own listings, and receive smart career advice â€” all in one place.

This project features a modern 3D UI, secure authentication with email verification, and an AI assistant powered by OpenAI or local models.

---

## Features

- User registration with email verification (via token and link)
- Login with secure JWT authentication
- Password reset via email
- AI assistant for career guidance and support
- Post and manage job listings
- Post and manage products for sale (with image uploads)
- Upload images and files using Cloudinary
- Modern 3D UI using React Three Fiber
- Fully responsive for all screen sizes

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Three.js (via React Three Fiber)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT for authentication
- Nodemailer for email sending
- Cloudinary for media hosting
